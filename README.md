# npm scripts
command|説明
---|---
yarn run|タスク一覧を表示する
yarn run watch|全てのタスクをwatchしながら実行する
yarn run css|sassをコンパイルする(+autoprefixir、cssnano)
yarn run js|coffeescriptをコンパイルする
yarn run bs|browserSyncを実行する
